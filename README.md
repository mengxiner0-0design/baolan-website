# BAOLAN BL Foodservice Website

这是一个静态网站版本，适合直接上传到 GitHub Pages / 服务器。

## 文件结构
- `index.html`：网站主页面
- `styles.css`：网站样式，蓝白 Apple 风格，高级极简布局
- `script.js`：滚动进入动画与产品筛选
- `images/`：独立图片资源文件夹
  - `bl-logo.png`：你上传的 LOGO
  - `hero-supply-system.png`：主视觉产品系统图
  - 11 张独立产品图片：手套、湿巾、围裙、打包袋、蒸笼纸、桌布、餐巾纸、餐包、腰封、标签、一次性烤炉

## 使用方式
直接打开 `index.html` 即可预览。
如果要部署到 GitHub Pages，把整个文件夹内容上传到仓库根目录即可。

## 注意
网站没有用整张大图压成页面，产品图都在 `images/` 文件夹中独立调用，后续可以替换为真实拍摄图或 AI 精修产品图。
