# BeautyPro Supply - Richer Website Version

This is a richer static website version for a beauty and salon disposable supplies business.

## What changed

- More sections and more commercial content
- More visible product imagery throughout the site
- Stronger category storytelling
- Procurement matrix section
- Packaging story section
- FAQ section
- Process section
- Contact section with mailto form logic
- Still deployable directly to GitHub Pages

## Files

- `index.html`
- `style.css`
- `script.js`
- `assets/products/*.jpg`

## Deploy to GitHub Pages

1. Upload all files to a GitHub repository.
2. Make sure `index.html` is in the root.
3. In GitHub: **Settings > Pages**
4. Set source to the main branch root.
5. Save and wait for deployment.
