# 餐饮一站式采购官网静态版

这是一个可直接打开的静态网站版本，包含：

- `index.html`：首页结构与内容
- `assets/css/styles.css`：响应式页面样式
- `assets/js/main.js`：移动端菜单、滚动动画、询价表单弹窗
- `assets/images/`：网站配图与裁切后的产品图

## 使用方法

1. 解压文件夹。
2. 双击打开 `index.html` 即可预览。
3. 后续可以把 `assets/images/` 里面的图片替换成真实产品图或工厂图。
4. 表单目前是静态演示版，后续可以接入邮箱、WhatsApp、企业微信机器人或 CRM。

## 当前网站定位

餐饮耗材一站式采购：手套、湿巾、围裙、打包袋、蒸笼纸、桌布、餐巾纸、餐包、腰封、标签、一次性烤炉等产品。
